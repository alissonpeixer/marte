# marte

### Start Angular-APP
```bash 
$ npm install -g @angular/cli
```
```bash 
$ npm i  --legacy-peer-deps
```
```bash 
$ ng serve
```


#### OBS Fix to erro `ENOSPC`
`sudo sysctl fs.inotify.max_user_watches=524288`

#### Depedencie 
Node >= 16 | AngularCli 15
